### C and C++ Language server support via clangd for [Xed-Editor](https://github.com/Xed-Editor/Xed-Editor)

**i do need some other features to be worked out 
like full completion support and functionality 
please contribute if you know how to fix.**

just a simple extension that installs clangd for xed.
you can get this extension from the extension store of xed or you can build it like this:
```
./compileRelease
```
or:
```
./compileDebug
```
Oh and dont forget to install the actual lsp 
by going to the settings --> editor --> Manage language servers --> and click the download button besides the toggle. 
see anything wrong just make a pr at https://github.com/pnthrr/xed-clangd .
